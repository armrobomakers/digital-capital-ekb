# Лендинг «Цифровой капитал» — Екатеринбург, 13 июня

Готовый лендинг под Vercel на Next.js.

## Как запустить локально

```bash
npm install
npm run dev
```

Откройте: http://localhost:3000

## Как загрузить на Vercel

1. Загрузите папку проекта в GitHub.
2. В Vercel нажмите **Add New Project**.
3. Выберите репозиторий.
4. Framework Preset: **Next.js**.
5. Deploy.

## Что нужно заменить перед публикацией

- В блоке `Место проведения` заменить текст адреса площадки.
- В форме регистрации заменить `https://formspree.io/f/your-form-id` на реальный обработчик формы.
- При необходимости заменить список спикеров, тарифы и программу в `app/page.tsx`.

## Основные файлы

- `app/page.tsx` — контент лендинга.
- `app/globals.css` — весь дизайн и адаптив.
- `app/layout.tsx` — SEO title/description.
